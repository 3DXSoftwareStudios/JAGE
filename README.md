# JAGE – Just Another Game Engine

JAGE is an attempt to create a new open‑source game engine from scratch.  
The editor is primarily developed in **Java**, while the runtime is designed to be **modular** and portable across different devices.

---

## How to Compile

1. Install **JDK 17 or higher**  
2. Clone this repository or download and extract the ZIP  
3. Navigate to the project root  
4. Run:

   ```bash
   ./gradlew run
